# FontHacker

## 1. 프로젝트 개요
FontHacker는 CNN 기반 손글씨 특징 추출과 유사도 비교를 통해
필기체를 분석·검증하는 웹 애플리케이션입니다.
단순 문자 인식이 아닌, 작성자 유사도 판단을 목표로 설계되었습니다.

## 2. 문제 정의
- 사용자 손글씨 입력 데이터는 획 굵기, 크기, 기울기 등 편차가 큼
- 단순 분류 모델로는 안정적인 비교가 어려움
- 실제 서비스 환경을 고려한 전처리 및 검증 로직 필요

## 3. 해결 방법
- 입력 이미지 품질 검증 및 정규화 전처리
- Siamese CNN을 활용한 특징 벡터 추출
- 코사인 유사도를 이용한 손글씨 간 유사도 비교

## 4. 시스템 구조
[입력] → [전처리] → [CNN 특징 추출] → [유사도 계산] → [웹 결과 출력]

## 5. 사용 기술
- Python, TensorFlow / Keras
- OpenCV, NumPy
- Flask, Docker
- HTML / CSS / JavaScript

## 6. 역할
- 프로젝트 기획 및 팀장
- 모델 구조 설계 및 전처리 로직 구현
- 웹 연동 및 전체 파이프라인 통합
